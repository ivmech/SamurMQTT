# SamurMQTT

### REQUIREMENTS

* RPi.GPIO
* Python 2.7 to 3.6.

### INSTALL

If you like to clone from source, you can do it like so:

```bash
git clone https://github.com/ivmech/SamurMQTT.git
```

If you have downloaded the source code:

    cd SamurMQTT
    sudo python setup.py install

### RUNNING TESTS

    samur_mqttd.py
